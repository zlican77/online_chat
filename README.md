# online_chat
a demo web-online-chat-page using golang

## YOU should first run the serve and then open the page
as follow

# to init
~~~
go mod init chat
~~~

# to require github
~~~
go get github.com/gorilla/websocket
go get github.com/gorilla/mux
~~~

# to build
~~~
go build
~~~

# to conduct
~~~
./chat.exe
~~~
